# AGENTS.md

## Project

EcoStat AI Real Data

## Goal

Build a simple web app for checking real environmental data and turning it into plain-language insights.

## Current App Type

Standalone HTML, CSS, and JavaScript.

This app is designed to run on GitHub Pages without Node.js, npm, backend services, databases, or API keys.

## Development Rules

- Start with a working MVP.
- Keep the interface simple.
- Make it easy for non-experts.
- Do not overbuild.
- Prefer public data APIs that work in the browser.
- Keep advanced statistics, AI, and backend work for later versions.

## Current Features

- Dashboard layout
- Sidebar navigation
- Real location search
- Current weather data
- Current air quality data
- KPI cards
- Basic pollutant chart
- Plain-language summary
- Editable data log
- Export data log as CSV
- Source links

## Data Sources

- Open-Meteo Geocoding API
- Open-Meteo Weather Forecast API
- Open-Meteo Air Quality API

## UI Style

Clean, modern, environmental, simple.
Inspired by dashboard tools, Power BI, and Excel.
