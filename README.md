# EcoStat AI Real Data

EcoStat AI Real Data is a standalone environmental data checker.

It fetches current weather and air quality data directly in the browser using public Open-Meteo APIs.

## What It Checks

- Location search
- Current weather
- Temperature
- Humidity
- Rain and wind
- US AQI
- European AQI
- PM2.5
- PM10
- Ozone
- Nitrogen dioxide
- Sulphur dioxide
- Carbon monoxide
- Editable real-data log
- CSV export

## Run

Open `index.html` in a browser.

For GitHub Pages, upload this folder to a repository and enable Pages from the `main` branch root.

## Data Sources

- Open-Meteo Weather Forecast API
- Open-Meteo Air Quality API
- Open-Meteo Geocoding API

## Scope

This is an MVP for real environmental data checking. It follows the project rules in `AGENTS.md`: keep the app simple, useful for non-experts, and working without backend services.

It is useful for screening and education, not official compliance reporting.
